## How to capture

1. Run the app: `npm run dev` (or use the live site).
2. Open each tab in your browser at a comfortable width (~1200px).
3. Take a screenshot:
   - **macOS:** `⇧⌘4` then drag (or `⇧⌘4` then Space to grab a whole window).
   - **Windows:** `Win+Shift+S`.
4. Save the PNG into this folder with the name from the table above.
5. Uncomment the corresponding line in the root README's **Screenshots** section.

Tip: a fixed browser width and the dark theme keep the set looking consistent.
